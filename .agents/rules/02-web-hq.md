# Web HQ AI Rules (`apps/hq-web`)

## 1. Framework & Rendering
- **Next.js**: Use Next.js 14+ with React Server Components (RSC).
- **Data Fetching**: Fetch data natively in server components whenever possible.

## 2. Styling & UI
- **CSS**: Tailwind CSS v4.
- **Animations**: Framer Motion for smooth micro-interactions and transitions.
