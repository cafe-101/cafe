# Agent Guidelines

## Agent Roles and Responsibilities
- Agents working in this module are responsible for creating, modifying, and maintaining the Expo-based mobile application for customers.
- Ensure all code strictly follows React Native and Expo best practices.

## Collaboration Guidelines
- Communicate clearly about module boundaries and dependencies.
- Make small, atomic commits and document changes thoroughly.
